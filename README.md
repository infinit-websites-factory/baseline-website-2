# Dealer Website

A modern dealer website for publishing vehicle inventory and capturing lead details.

## Features

- Vehicle inventory display
- Lead capture forms
- Responsive design
- Modern UI/UX

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/alx-tech/infinit-demo-website-es.git

# Navigate to the project directory
cd infinit-demo-website-es

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

### Local Development

Run the development server with auto-reloading:

```sh
npm run dev
```

### Building for Production

```sh
npm run build
```
